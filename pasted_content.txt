# Prompt for Lovable.dev - Portfolio Site for AI/ML Engineer

## Site Structure

- Dark/Light mode toggle in the top-right corner
- Footer with social links and copyright info

- **Primary Color Scheme**: Dark theme with deep navy (#0F172A) background, slate gray (#334155) for secondary elements, and teal (#0D9488) for accents and interactive elements
- **Typography**: Modern sans-serif fonts (Inter for headings, IBM Plex Sans for body text, JetBrains Mono for code snippets)
- **Spacing**: Generous whitespace for readability
- **Animations**: Subtle micro-interactions and smooth transitions between sections/pages
- **Layout**: Three-column grid on desktop that collapses appropriately on mobile devices
